读取当前项目的真实代码和配置，生成标准格式的纯文本项目介绍。

## 步骤

1. 读取 package.json 获取完整依赖
2. 读取 prisma/schema.prisma 获取数据库结构
3. 读取 CLAUDE.md 或 README.md 获取项目描述
4. 读取 app/ 目录结构了解功能模块

## 严格按以下格式输出，纯文本，无任何 markdown 符号

简介（50字左右，一句话说清楚产品定位和核心价值）

技术栈: 技术1、技术2、技术3...（从 package.json 真实提取，写在一行）

1.亮点一（不超过30字，精炼突出技术要点）
2. 亮点二（不超过30字，精炼突出技术要点）
3. 亮点三（不超过30字，精炼突出技术要点）
...

## 注意事项

- 只写真实存在于项目中的技术，绝对不捏造
- 技术栈写在一行，顿号分隔，不分类不换行
- 每条亮点严格不超过 30 字，突出具体技术实现
- 输出全程不含任何 markdown 格式符号
