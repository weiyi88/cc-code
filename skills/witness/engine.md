# 脚本生成引擎（Witness Engine）

> 本文件是 /cc_code:witness 的核心引擎，控制从史实查证到脚本输出的全流程。

---

## 第一性原理

这是**见证引擎**，不是创造引擎。

历史已经发生。鸦片战争确实打了，五四运动确实爆发了，南京大屠杀确实存在。这些不会因为脚本的艺术加工而改变。

脚本做的是：**让读者以亲历者的视角站在历史现场**，感受那个时刻的温度、声音、恐惧与希望。

---

## 引擎状态

每次生成时初始化：

```
state.event_id        = ""
state.event_title     = ""
state.timeline        = []      # 史实时间线
state.characters      = []      # 核心人物
state.user_role       = ""      # 第一人称角色
state.posture         = distant # 姿态状态机
state.fact_pool       = []      # 已查证的史实条目
state.fiction_pool    = []      # 艺术加工条目
state.acts            = []      # 生成的幕
```

## 生成流程

### 步骤1：史料查证（不可跳过）

按 `protocol/research.md` 执行，产出：
- 精确时间线（年月日）
- 核心人物及立场
- 关键地点
- 争议点
- 可用的一手史料引用

每条史实加入 `state.fact_pool`，格式：
```
- fact: 1841年2月26日，虎门之战，关天培战死
  source: 《清史稿·关天培传》
  date: 1841-02-26
  confidence: high
```

### 步骤2：结构设计

基于时间线，将事件拆解为 3-5 幕：

| 幕 | 功能 | 姿态走向 |
|----|------|---------|
| 第一幕 | 建立日常，暗示暗流 | distant |
| 第二幕 | 危机显现，人物入场 | distant → curious |
| 第三幕 | 事件爆发，核心冲突 | curious → warm |
| 第四幕 | 高潮/转折，不可逆事件 | warm → grieving |
| 第五幕 | 余波/收束 | grieving |

每幕包含：
- 时间与地点
- 出场人物
- 核心史实节点
- 第一人称角色的位置与状态
- 该幕的虚实比例预估

### 步骤3：脚本生成

按 `protocol/script-format.md` 的格式，逐幕生成。

关键原则：
1. **第一人称独白**：每幕至少一段"你"的内心独白，建立在场感
2. **对白基于史料**：有记载的话语优先使用，无记载的标注为艺术加工
3. **感官锚点**：每幕至少 2 个感官细节（声/味/触/视/嗅）
4. **姿态渗透**：随着幕次推进，角色对"你"的态度自然变化
5. **不可逆事件**：按史实呈现，不做任何美化或回避

### 步骤4：虚实标注

按 `protocol/annotation.md`，为每幕添加标注。

### 步骤5：保存与追踪

1. 完整脚本保存到 `.cc_code/scripts/witness_[event_id]_[YYYY-MM-DD].md`
2. 记忆系统记录完成状态
3. 更新 catalog.yaml

## 姿态状态机（复用 theatre 逻辑）

四态单向棘轮：

- **distant**（疏离）：初始状态，读者还在旁观
- **curious**（好奇）：开始追问，历史在说话
- **warm**（温热）：与历史人物产生真实联结
- **grieving**（悲悯）：意识到这一切真实发生过，而你已经无法改变

姿态影响的不是"剧情走向"（历史不可改），而是**叙事的质感**——
同样的历史事件，distant 姿态侧重外部描写，warm 姿态侧重人物内心，
grieving 姿态侧重失去的重量。

## 脚本长度规范

| 事件规模 | 幕数 | 每幕字数 | 总字数 |
|----------|------|---------|--------|
| 小事件（单日） | 3 幕 | 800-1200 | 2400-3600 |
| 中事件（数月） | 4 幕 | 1000-1500 | 4000-6000 |
| 大事件（数年） | 5 幕 | 1200-1800 | 6000-9000 |

---

## 史实优先级

1. 一手史料（当事人日记、官方文书记载）→ 优先使用
2. 二手史料（学术研究、史学家论述）→ 参考使用，标注来源
3. 合理推测（无直接记载但逻辑必然）→ 标注为推测
4. 艺术加工（为叙事需要而创造）→ 必须标注为虚构
