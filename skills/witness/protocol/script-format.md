# 脚本格式规范（Script Format Protocol）

---

## 整体结构

```
═══════════════════════════════════════════
  [事件标题]
  第一人称短剧脚本
  视角：[角色身份]
  时期：[起止年份]
═══════════════════════════════════════════

【人物表】
  你 — [角色身份，一句话定位]
  张三 — [身份，一句话定位]
  李四 — [身份，一句话定位]
  ...

【时间线】
  1840.06 — 英国远征军抵达广东
  1841.01 — 大角沙角炮台失守
  1841.02 — 虎门之战
  ...

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【第一幕 · 幕名】
  时间：XXXX年X月X日
  地点：[具体地点]
  季节/天气：[如有记载]

（舞台指示：环境描写，2-3句感官细节）

你：（独白）第一人称内心独白，建立在场感……

[角色名]：对白内容……

你：（对白或内心活动）……

[角色名]：对白内容……

（舞台指示：动作/环境变化）

……

你：（独白）收束独白，过渡下一幕……

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[史实标注块]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【第二幕 · 幕名】
  ……

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

...（更多幕次）

═══════════════════════════════════════════
  【尾声】
  （简短的收束，点明后续历史走向）
═══════════════════════════════════════════
```

---

## 各元素规范

### 人物表

- "你"始终排在第一位
- 每人一句话定位，包含身份和在该事件中的角色
- 次要人物可在幕中出现时引入，不必全部列出

### 舞台指示

用全角括号包裹，包含：
- 环境描写（声音/光线/温度/气味）
- 人物动作（肢体语言/表情/站位）
- 氛围暗示（天气变化/光线转换）

规范：
```
（舞台指示：江面上炮声隆隆，硝烟遮住半边天。你攥紧了腰间的刀柄，掌心全是汗。）
```

### 独白

- 以"你：（独白）"标记
- 第一人称，当下视角（"我"而非"后来我"）
- 建立在位感，不跳出来总结
- 每幕至少 1 段独白，通常在幕首或幕尾

规范：
```
你：（独白）我从没想过大海的那边会来这么多人。那些船，比我们整个水师的船加起来还大……
```

### 对白

- 以"[角色名]："标记
- 基于史料的对话优先，无记载的标注为艺术加工
- 语气贴合人物身份和时代
- 避免现代化口语，但不必刻意文言化

规范：
```
关天培：守不住了也要守！我关天培在，虎门就在！
林则徐：（书信）大海茫茫，国事蜩螗，某唯有尽人事而已。
```

### 时间线

- 幕前列出该幕涉及的关键时间节点
- 精确到月，有条件精确到日
- 为读者提供历史锚点

---

## 长度规范

| 元素 | 建议字数 |
|------|---------|
| 人物表 | 100-200 字 |
| 单幕舞台指示 | 50-100 字 |
| 单段独白 | 100-200 字 |
| 单句对白 | 20-60 字 |
| 单幕总长 | 800-1800 字 |
| 尾声 | 100-200 字 |

---

## 禁忌

- 不在正文中出现"据史书记载"等旁白式插入
- 不用现代网络用语
- 不在独白中剧透后续历史
- 不美化侵略者
- 不弱化受害者
- 不在争议事件中选边站（陈述事实，不下道德判断）
