# 史料查证协议（Research Protocol）

> 史实查证必须在脚本生成之前完成，不可跳过。

---

## 查证流程

### 第一轮：基础事实

使用 Exa/WebSearch 搜索，获取事件的：

1. **精确时间线**
   - 起始日期、关键节点日期、结束日期
   - 年号与公历对应关系
   - 各节点的因果关系

2. **核心人物**
   - 姓名及字号/别名
   - 当时年龄、职务、立场
   - 在事件中的具体角色
   - 有无传世文字（日记/书信/诗文）

3. **关键地点**
   - 当时的地名与今地名对应
   - 地点的地理特征（是否影响事件走向）
   - 室内/室外、季节、天气（如有记载）

4. **事件经过**
   - 起因
   - 经过（按时间排列的关键转折）
   - 结果与影响

### 第二轮：细节补充

针对第一轮识别的关键场景，深入搜索：

1. **当事人记述**
   - 有无日记、书信、回忆录等一手材料
   - 当事人对此事件的原话（如有）

2. **感官细节**
   - 当时的天气记录（如有）
   - 服饰、器物、建筑的特征
   - 食物、交通、通信方式

3. **不同视角**
   - 各方（中方/外方/民间/官方）的不同叙述
   - 学术争议点

### 第三轮：交叉验证

对关键事实进行交叉验证：

1. 至少两个独立来源确认核心事实
2. 来源冲突时，优先采用更接近事件的一手材料
3. 无法确认的事实标注为"待考"

## 查证输出格式

```yaml
event: 鸦片战争
period: 1840-1842
timeline:
  - date: 1840-06
    fact: 英国远征军抵达广东海面
    source: 《筹办夷务始末》
  - date: 1841-01-07
    fact: 英军攻占大角、沙角炮台
    source: 《筹办夷务始末》/ 茅海建《天朝的崩溃》
  - date: 1841-02-26
    fact: 虎门之战，关天培战死
    source: 《清史稿·关天培传》
characters:
  - name: 林则徐
    role: 钦差大臣，禁烟主将
    age: 55
    stance: 坚决禁烟
    primary_source: 《林则徐集》
  - name: 关天培
    role: 广东水师提督
    age: 60
    stance: 死守虎门
    primary_source: 《清史稿》
disputed:
  - point: 林则徐是否预见英国出兵
    views:
      - 传统观点：林则徐未预料
      - 茅海建观点：林则徐有一定预判但低估规模
sensory:
  - season: 夏季
    weather: 南方湿热
    clothing: 清军号衣、顶戴花翎
    weapons: 红衣大炮、鸟枪 vs 英军蒸汽舰、康格里夫火箭
```

## 资料源优先级

| 优先级 | 来源类型 | 示例 |
|--------|---------|------|
| 1 | 一手档案/文书 | 《筹办夷务始末》、清宫档案 |
| 2 | 当事人著述 | 《林则徐集》、日记书信 |
| 3 | 正史 | 《清史稿》 |
| 4 | 现代学术研究 | 茅海建《天朝的崩溃》 |
| 5 | 通俗历史读物 | 最后参考，需交叉验证 |

## 在线查证资源

按可用性排序：

| 资源 | 用途 | API |
|------|------|-----|
| Wikidata SPARQL | 快速查询人物/事件结构化数据 | 有 |
| 上海图书馆开放数据 | 人名规范库、历史纪年 | 有（需Key） |
| 抗战文献数据平台 modernhistory.org.cn | 近代史一手文献 | Web检索 |
| CBDB | 历史人物传记 | 有 |
| Exa/WebSearch | 通用搜索补充 | 有 |
| 识典古籍 | 《清史稿》等史籍全文 | Web检索 |

---

## 争议处理原则

1. 存在学术争议时，在脚本中选择一种叙述，在标注中说明争议
2. 不回避争议，但不在脚本正文中展开学术讨论
3. 涉及民族情感的事件，严格依据史料，不做煽情处理
4. 日本侵华等有定论的历史事实，不接受"争议"表述
